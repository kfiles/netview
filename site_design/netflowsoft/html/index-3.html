OPTIMIZING NETWORK PERFORMANCE MANAGEMENT

To optimize network application performance and improving network service delivery NetView fulfills the following key roles:

    Monitor

  • Simplified, intuitive, enterprise-wide and network element views of the network.
  • Leverage existing infrastructure instrumentation and passive monitoring techniques.

    • Baseline

  • Dynamic baselines minimize false positive alerts and enable management by exception.

    • Analyze

  • Network Behavior Analysis (NBA).
  • Application Profiling.
  • Root cause determination aiding in network performance optimization and security breach detection.

    • Report

  • Real-time and long-term reporting of all network communications.
  • Trend analysis.
  • Critical aid for capacity planning and traffic engineering.
